Fleetio Works

FLEETIO has two access levels, one is for admin and it is a web based application and one is for drivers and it is a mobile based application. Admin can login to system by using admin username and his/her password and Drivers can log into system using their usernames and passwords. Home pages of each users are different according to their needs and requirements.

Access Level – Admin

Admin hold the full authority of the web application. He/She can access system and view all vehicles locations and can view locations of drivers through a google map.

Admin also capable of sending notifications to drivers what where houses they should go to.

Admin authorize the ware houses assigning process and also can view customers complains to take actions.

Admin can select particular vehicle and view its history up to a week. There is login interface for the admin inside the driver’s mobile application and through that admin can log into application and he can change the assigned vehicle number of the mobile phone and assign mobile phone to another vehicle.

Best and newest part of this application is live streaming module. Admin has a authority to watch the particular vehicle’s delivering cargo by using IP camera.


Access Level – Driver

Driver is the person who mostly involve with the mobile application. This application has driver required functions.

QR reader, Google map loader, view notification and Job done button. When driver got the ware houses, he go to that pace and should read the provided QR code using mobile and inside that QR code it holds the destinations.

When reads the QR then application pick up the destinations and put that on google map and show the shortest path to the driver. He can fallow that path.

After he done the job submit job done button and it will display to admin now he is available for another delivery.


Access Level – Customer

Customer has a complain page to provide complains that he/she has. Customer has to provide name, telephone number, Email, and massage. It will display to the admin and if admin is not available admin can access that massage later.
